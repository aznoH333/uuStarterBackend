### How to auth

1. http://localhost:3001/auth/signin (google login)
2. http://localhost:3001/custom-token (get you cutom token for 15 mins) 
3. http://localhost:3001/api/me Needs token in json
4. http://localhost:3001/api/admin Needs token in json
5. http://localhost:3001/login-basic Login for basic user, returns token atd voe.
